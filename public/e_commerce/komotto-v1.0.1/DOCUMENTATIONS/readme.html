What you should know

his theme is built using HTML-framework Bootstrap from Twitter. Theme uses a Jquery plugins third party, and .LESS for easy and quick editing CSS. To change the content of the template, you will need additional software, such as: Brackets (by Adobe) with the plugin LessAutocompile to edit HTML and CSS, Photoshop. You can also use their analogues.

Parallax

To work with the parallax effect, connect the jquery plugin: jQuery.TDParallax.js
Insert a block with parallax into the wrapper, give it a class: parallax-wrap

Customize the plugin through the data attributes.
data-parallax-image sets the background image
data-speed-direction speed
data-default-pos Starting position
data-parallax-block It ignores the image and creates a parallax effect for the block

Timer

Assign a class timer to the block.
Set the timer expiration through the date attribute: data-timer-date

The following elements will take values depending on the class assigned to them
tdtimer-d - days
tdtimer-h - hours
tdtimer-m - minutes
tdtimer-s - seconds
